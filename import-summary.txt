Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* AndroidManifest.xml => AMaze12/src/main/AndroidManifest.xml
* assets/ => AMaze12/src/main/assets
* libs/android-support-v4.jar => AMaze12/libs/android-support-v4.jar
* res/ => AMaze12/src/main/res/
* src/ => AMaze12/src/main/java/
* src/.DS_Store => AMaze12/src/main/resources/.DS_Store
* src/edu/wm/cs/.DS_Store => AMaze12/src/main/resources/edu/wm/cs/.DS_Store
